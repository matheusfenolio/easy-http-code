# easy-http-code

## Sumary
Builded to make easier HTTP codes use.

### Javascript
```javascript
const { HttpStatus, HttpMessage } = require('easy-http-code');

console.log(HttpStatus.NO_CONTENT);
console.log(HttpMessage(204))
```

### Typescript
```javascript
import { HttpStatus, HttpMessage } from 'easy-http-code';

console.log(HttpStatus.NO_CONTENT);
console.log(HttpMessage(204))
```
